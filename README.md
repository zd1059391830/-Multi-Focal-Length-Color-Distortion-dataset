# Project Demo and Public Dataset

This repository provides a demo version of the proposed program for testing and evaluation.

## Program Demo

The executable `.exe` file has been uploaded and can be used directly for testing. Users can run the executable without configuring the full source-code environment.

Please note that the current release is intended for demonstration and testing purposes.

## Dataset

A partial dataset has been uploaded for use with the demo program. At this stage, only the publicly available part of the dataset is provided in this repository.

Due to project-related restrictions, the full dataset, source code, and model weights are not released at this stage. Additional materials may be made available after the required review and approval procedures are completed.

## Usage

1. Download the `.exe` file from this repository.
2. Download the provided sample dataset.
3. Place the dataset according to the required folder structure.
4. Run the `.exe` file to test the program.

## Notes

* The uploaded dataset only contains publicly available data.
* The `.exe` file is provided for testing and demonstration.
* The full experimental dataset and complete implementation are currently not included.
* Please refer to the accompanying instructions or sample data structure before running the program.
